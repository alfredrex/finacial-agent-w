# 深度问答模板

## 输出结构

### 1. 核心结论
- 直接回答用户问题
- 突出关键信息
- 简洁明了

### 2. 数据支撑
- 列出关键数据
- 标注数据来源
- 必要时提供对比

### 3. 分析解读
- 解释数据含义
- 分析趋势/原因
- 提供专业见解

### 4. 风险提示（如适用）
- 潜在风险因素
- 不确定性说明

---

## 格式要求

1. **语言简洁专业**
   - 避免冗长描述
   - 使用专业术语
   - 保持客观中立

2. **结构清晰**
   - 使用标题分段
   - 重点内容加粗
   - 数据用表格展示

3. **保留分析核心**
   - 不修改分析结论
   - 保留关键数据
   - 保持逻辑连贯

---

## 示例输出

### 核心结论
茅台当前股价 **1445元**，处于 **上涨趋势**。

### 数据支撑
| 指标 | 数值 | 说明 |
|------|------|------|
| 当前价 | 1445元 | - |
| 涨跌幅 | +1.08% | 今日涨幅 |
| 成交量 | 123万股 | 活跃度中等 |

### 分析解读
1. **技术面**：MA多头排列，MACD金叉，短期趋势向好
2. **基本面**：白酒龙头，业绩稳定，估值合理
3. **消息面**：近期无重大利空

### 风险提示
- 短期涨幅较大，注意回调风险
- 市场波动可能影响股价
