# 趋势周期分析 Reference

## 分析目标
分析长/中/短周期和趋势延续性

---

## 分析框架

### 1. 周期划分
- **长期趋势（>1年）**
  - 周线/月线级别
  - 基本面驱动
  - 主要趋势方向

- **中期趋势（1-6月）**
  - 日线级别
  - 行业/板块轮动
  - 次级趋势方向

- **短期趋势（<1月）**
  - 小时线/分钟线
  - 情绪/事件驱动
  - 波动方向

### 2. 趋势识别
- **上涨趋势特征**
  - 高点不断创新高
  - 低点不断抬高
  - 均线多头排列
  - 回调不破关键支撑

- **下跌趋势特征**
  - 低点不断创新低
  - 高点不断降低
  - 均线空头排列
  - 反弹不破关键压力

- **震荡趋势特征**
  - 高点低点在区间内
  - 均线交叉缠绕
  - 支撑压力有效

### 3. 周期节点判断
- **趋势起点**
  - 突破关键位置
  - 量能配合
  - 多周期共振

- **趋势中继**
  - 回调/反弹后延续
  - 量能萎缩后再次放大
  - 技术形态修复

- **趋势末端**
  - 量价背离
  - 技术指标钝化
  - 多周期冲突

### 4. 趋势强度评估
- **强趋势**
  - 多周期同向
  - 量能持续配合
  - 回调/反弹幅度小

- **弱趋势**
  - 多周期冲突
  - 量能不配合
  - 回调/反弹幅度大

## 注意事项

1. 长期趋势优先级最高，短期服从长期
2. 多周期共振时趋势最可靠
3. 关注周期转换信号
4. 不同投资风格关注不同周期
