# 事件/新闻驱动分析 Reference

## 分析目标
分析消息对股价的影响（利好/利空、短期/长期）

---

## 分析框架

### 1. 事件分类
- **公司层面**：业绩公告、重组并购、股权变动、高管变动
- **行业层面**：政策变化、行业监管、技术突破
- **宏观层面**：经济数据、货币政策、国际形势

### 2. 影响评估
- **利好/利空判断**
  - 利好：提升业绩预期、改善经营环境、增强竞争力
  - 利空：降低业绩预期、恶化经营环境、削弱竞争力
  - 中性：影响不明确或已充分预期

- **影响程度**
  - 重大：可能改变公司基本面
  - 中等：影响短期业绩或情绪
  - 轻微：影响有限

- **影响周期**
  - 短期（1-5天）：情绪驱动，快速反应
  - 中期（1-3月）：业绩影响逐步体现
  - 长期（>3月）：基本面改变

### 3. 市场反应预测
- **已反应程度**
  - 消息公布前已大涨 → 利好可能已兑现
  - 消息公布前已大跌 → 利空可能已消化
- **预期差**
  - 超预期 → 可能继续上涨/下跌
  - 符合预期 → 影响有限
  - 不及预期 → 可能反向反应

### 4. 风险评估
- **消息真实性**：官方公告 vs 市场传闻
- **执行风险**：政策落地、重组成功的不确定性
- **市场情绪**：当前市场风险偏好

---

## 注意事项

1. 区分事实与观点，以官方公告为准
2. 避免过度解读单一事件
3. 考虑市场当前情绪和风险偏好
4. 关注事件的持续性和后续发展
