# 技术面分析 Reference

## 分析目标
通过K线、均线、量能等技术指标判断股价走势

---

## 分析框架

### 1. 趋势判断
- **均线系统**
  - 多头排列：MA5 > MA10 > MA20 > MA60 → 上涨趋势
  - 空头排列：MA5 < MA10 < MA20 < MA60 → 下跌趋势
  - 交叉缠绕 → 震荡趋势
- **趋势强度**
  - 价格在所有均线上方 → 强势
  - 价格在所有均线下方 → 弱势

### 2. 动量指标
- **RSI（相对强弱指数）**
  - RSI > 70 → 超买，注意回调
  - RSI < 30 → 超卖，可能反弹
  - RSI 30-70 → 正常区间
- **MACD**
  - 金叉（DIF上穿DEA）→ 买入信号
  - 死叉（DIF下穿DEA）→ 卖出信号
  - 柱状图由负转正 → 动能转强

### 3. 波动指标
- **布林带**
  - 价格触及上轨 → 超买
  - 价格触及下轨 → 超卖
  - 布林带收窄 → 变盘在即
- **ATR（真实波动幅度）**
  - ATR增大 → 波动加剧
  - ATR减小 → 波动收敛

### 4. 成交量分析
- **量价关系**
  - 放量上涨 → 健康上涨
  - 放量下跌 → 恐慌抛售
  - 缩量上涨 → 上涨乏力
  - 缩量下跌 → 下跌动能不足
- **换手率**
  - >10% 活跃
  - 3-10% 正常
  - <3% 不活跃

### 5. 支撑压力位
- **支撑位**：前期低点、重要均线、成交密集区
- **压力位**：前期高点、重要均线、成交密集区

---

## 注意事项

1. 技术分析基于历史数据，不能预测未来
2. 多个指标要综合判断，不要单一依赖
3. 趋势判断优先级：长期 > 中期 > 短期
4. 成交量是重要验证指标
