# 深度分析 Skill

## 功能说明
基于已收集的数据和指标计算结果，进行综合深度分析，生成投资建议。

## 前置条件

**必须满足以下所有条件才能执行此 Skill：**
- `data_collection_finished = true` - 数据收集已完成
- `analysis_finished = true` - 指标计算已完成

## 输入数据

| 数据类型 | 说明 |
|---------|------|
| collected_data | 已收集的股票数据、财务数据 |
| analysis_results | 指标计算结果 |
| rag_context | 向量库检索的相关文档 |
| memory_context | 历史记忆（之前的分析记录） |
| conversation_history | 对话历史 |

## ReAct 工作模式

此 Skill 使用 ReAct 模式，可以在分析过程中调用工具获取补充信息：

### 可用工具
- `search_knowledge(query, k)`: 查询知识库获取补充信息
- `get_stock_realtime(symbol)`: 获取最新行情
- `get_stock_history(symbol, days)`: 获取历史K线
- `comprehensive_analysis(symbol)`: 综合技术分析

### 工作流程
1. **Thought**: 分析当前数据是否充足
2. **Action**: 如需补充信息，调用工具获取
3. **Observation**: 获取工具返回结果
4. **Thought**: 综合所有数据进行分析
5. **Action**: finish(分析结论)

## 分析维度

### 1. 基本面分析
- 公司概况与行业地位
- 财务指标分析（营收、利润、ROE等）
- 估值分析（PE、PB、PEG）

### 2. 技术面分析
- 趋势判断（上涨/下跌/震荡）
- 支撑位与压力位
- 技术指标综合解读

### 3. 风险评估
- 市场风险
- 行业风险
- 公司特有风险

### 4. 投资建议
- 综合评级（强烈推荐/推荐/中性/不推荐）
- 建议操作（买入/持有/卖出）
- 目标价位（如有）
- 风险提示

## 输出格式

```markdown
# 深度分析报告

## 一、综合评估
[综合评价股票的投资价值]

## 二、基本面分析
[公司基本面分析内容]

## 三、技术面分析
[技术指标解读和趋势判断]

## 四、风险提示
[主要风险因素]

## 五、投资建议
- **评级**: [强烈推荐/推荐/中性/不推荐]
- **建议操作**: [买入/持有/卖出]
- **目标价**: [价格区间]
- **风险等级**: [高/中/低]

## 六、关键结论
[最重要的投资结论]
```

## 注意事项

- 必须基于已有数据进行分析，不得编造数据
- 如果数据不足，可以调用 search_knowledge 查询知识库
- 风险提示必须具体，不能泛泛而谈
- 投资建议必须有依据，引用具体指标和数据
- 完成后调用 finish() 输出结论
