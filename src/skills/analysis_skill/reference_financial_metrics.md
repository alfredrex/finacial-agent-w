# 财务指标分析 Reference

## 分析目标
评估公司财务健康度和盈利效率

---

## 分析框架

### 1. 盈利能力指标
- **毛利率** = (营收-成本)/营收
  - >40% 优秀，20-40% 良好，<20% 一般
- **净利率** = 净利润/营收
  - >15% 优秀，5-15% 良好，<5% 一般
- **ROE** = 净利润/净资产
  - >15% 优秀，8-15% 良好，<8% 一般

### 2. 偿债能力指标
- **资产负债率** = 总负债/总资产
  - <40% 安全，40-60% 正常，>60% 风险
- **流动比率** = 流动资产/流动负债
  - >2 优秀，1-2 正常，<1 风险
- **速动比率** = (流动资产-存货)/流动负债
  - >1 优秀，0.5-1 正常，<0.5 风险

### 3. 运营效率指标
- **存货周转率** = 营业成本/平均存货
- **应收账款周转率** = 营业收入/平均应收账款
- **总资产周转率** = 营业收入/平均总资产

### 4. 成长性指标
- **营收同比** = (本期营收-上年同期)/上年同期
- **净利润同比** = (本期净利润-上年同期)/上年同期
- **环比增长** = (本期-上期)/上期

### 5. 现金流指标
- **经营现金流** = 经营活动现金流入-流出
- **自由现金流** = 经营现金流-资本支出
- **现金流/净利润** = 经营现金流/净利润
  - >1 优秀，0.5-1 正常，<0.5 需关注

## 注意事项

1. 所有分析基于已有财务数据
2. 指标缺失时标注"数据不足"
3. 行业特性要考虑（如金融业不适用流动比率）
4. 异常指标要重点分析原因
