# 基本面分析 Reference

## 分析目标
评估公司基本面质量，判断投资价值

---

## 分析框架

### 1. 公司质地分析
- **行业地位**：是否为行业龙头？市占率多少？
- **商业模式**：盈利模式是否清晰？可持续性如何？
- **竞争优势**：护城河是什么？品牌/技术/渠道/成本？
- **管理层**：管理层背景、治理结构、股权激励

### 2. 成长性分析
- **营收增长**：近3年营收增速及趋势
- **利润增长**：近3年净利润增速及趋势
- **增长驱动**：增长来源（内生/外延/行业红利）

### 3. 盈利能力分析
- **毛利率**：是否稳定？行业对比如何？
- **净利率**：盈利效率如何？
- **ROE**：净资产收益率水平及趋势

### 4. 估值分析
- **PE/PB**：当前估值水平，历史分位
- **同行对比**：与同业公司估值对比
- **安全边际**：当前价格是否具备安全边际

---

## 注意事项

1. 基于已有数据进行分析，不调用额外工具
2. 数据不足时明确标注，不做主观臆断
3. 结论要有数据支撑，逻辑清晰
4. 风险提示要具体，不要泛泛而谈
