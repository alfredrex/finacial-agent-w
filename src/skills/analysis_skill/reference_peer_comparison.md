# 同业对比分析 Reference

## 分析目标
与同行对比强弱，评估公司竞争力

---

## 分析框架

### 1. 选取对标公司
- 同行业龙头
- 业务相似公司
- 市值相近公司
- 选择3-5家对比

| 净利润 | XX亿 | XX亿 | XX亿 | XX亿 |

### 6. 竞争力评估
- **优势领域**：哪些指标领先同行
- **劣势领域**：哪些指标落后同行
- **差异化**：与同行的差异化竞争点

---

## 注意事项

1. 对标公司选择要合理，业务相似度要高
2. 行业平均数据要考虑样本代表性
3. 不同发展阶段的公司对比要谨慎
4. 关注公司特有因素（如新业务、转型等）
