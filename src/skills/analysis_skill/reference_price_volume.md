# 量价关系分析 Reference

## 分析目标
通过量能配合判断真假突破和趋势可靠性

---

## 分析框架

### 1. 量价基础关系
| 价格 | 成交量 | 信号 | 可靠性 |
|------|--------|------|--------|
| 上涨 | 放量 | 健康上涨 | 高 |
| 上涨 | 缩量 | 上涨乏力 | 中 |
| 下跌 | 放量 | 恐慌抛售 | 高 |
| 下跌 | 缩量 | 下跌动能不足 | 中 |
| 横盘 | 放量 | 变盘在即 | 高 |
| 横盘 | 缩量 | 继续横盘 | 高 |

### 2. 突破验证
- **真突破特征**
  - 放量突破（成交量明显放大）
  - 突破后回踩不破
  - 多次测试压力位后突破

- **假突破特征**
  - 缩量突破
  - 突破后快速回落
  - 没有经过充分整理

### 3. 趋势确认
- **上涨趋势确认**
  - 放量突破前高
  - 回调缩量
  - 再次放量上涨

- **下跌趋势确认**
  - 放量跌破前低
  - 反弹缩量
  - 再次放量下跌

### 4. 量能指标
- **换手率**
  - >20%：极度活跃，注意风险
  - 10-20%：非常活跃
  - 5-10%：活跃
  - 1-5%：正常
  - <1%：不活跃

- **量比**
  - >5：异常放量
  - 2-5：明显放量
  - 1-2：温和放量
  - <1：缩量

- **OBV（能量潮）**
  - OBV上涨 + 价格上涨 → 健康上涨
  - OBV下跌 + 价格上涨 → 上涨乏力
  - OBV上涨 + 价格下跌 → 底部积累

## 注意事项

1. 量在价先，成交量是重要先行指标
2. 关注异常放量，可能是变盘信号
3. 结合股价位置判断（高位放量 vs 低位放量）
4. 不同市场环境下量价含义可能不同
