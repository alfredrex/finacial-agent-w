# 简短报告模板

## 报告结构

### 1. 概述
- 一句话总结核心结论

### 2. 关键数据
- 3-5个核心指标
- 表格展示

### 3. 简要分析
- 2-3条要点

---

## 格式要求

- 总长度不超过 500 字
- 突出核心信息
- 省略详细分析

---

## 示例

### 概述
茅台当前股价1445元，短期趋势向好。

### 关键数据
| 指标 | 数值 |
|------|------|
| 股价 | 1445元 |
| 涨跌幅 | +1.08% |
| 市盈率 | 25倍 |

### 简要分析
1. 技术面多头排列，短期看涨
2. 基本面稳健，估值合理
3. 建议关注回调机会
