# 行业深度研究报告模板

## 报告结构

### 1. 行业概况
- 行业定义与分类
- 产业链分析
- 市场规模

### 2. 行业现状
- 发展阶段
- 竞争格局
- 主要玩家

### 3. 驱动因素
- 政策驱动
- 需求驱动
- 技术驱动

### 4. 投资机会
- 细分赛道机会
- 龙头企业机会
- 成长性企业机会

### 5. 风险提示
- 政策风险
- 竞争风险
- 技术风险

### 6. 投资建议
- 行业评级
- 重点标的
- 投资策略

---

## 格式要求

1. **宏观视角**：从行业整体出发
2. **数据支撑**：提供行业数据
3. **逻辑清晰**：投资逻辑明确
4. **标的明确**：推荐具体标的

---

## 示例

### 1. 行业概况

**白酒行业**

| 项目 | 内容 |
|------|------|
| 市场规模 | 6000亿 |
| 企业数量 | 2000+ |
| CR5 | 30% |

**产业链**：
- 上游：粮食、包装
- 中游：白酒生产
- 下游：经销商、终端

### 2. 行业现状

#### 发展阶段
- 成熟期，增速放缓
- 结构性机会为主

#### 竞争格局
- 高端：茅台、五粮液、泸州老窖
- 次高端：汾酒、洋河
- 中低端：区域品牌

### 3. 驱动因素

| 驱动因素 | 说明 |
|---------|------|
| 政策 | 消费升级政策支持 |
| 需求 | 中产阶级扩大 |
| 技术 | 数字化营销 |

### 4. 投资机会

#### 细分赛道
- 高端白酒：确定性最强
- 次高端：成长性最好

#### 重点标的
| 标的 | 逻辑 |
|------|------|
| 茅台 | 龙头，确定性 |
| 五粮液 | 估值修复 |
| 汾酒 | 成长性 |

### 5. 风险提示

| 风险 | 说明 |
|------|------|
| 政策 | 消费税改革 |
| 竞争 | 行业竞争加剧 |
| 消费 | 消费降级 |

### 6. 投资建议

**行业评级**：看好

**投资策略**：
1. 长期持有龙头
2. 关注次高端成长
3. 控制仓位，分散配置
