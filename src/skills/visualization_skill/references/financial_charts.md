# 金融图表最佳实践

## 图表类型选择指南

### 按数据类型选择

| 数据类型 | 推荐图表 | 适用场景 |
|---------|---------|---------|
| 时间序列 | K线图、趋势图 | 股价走势、指标变化 |
| 对比数据 | 柱状图、条形图 | 公司对比、行业对比 |
| 占比数据 | 饼图、环形图 | 资产配置、收入结构 |
| 分布数据 | 直方图、箱线图 | 收益分布、波动分析 |
| 相关数据 | 散点图、热力图 | 因子相关性分析 |

### 按分析目的选择

| 分析目的 | 推荐图表 |
|---------|---------|
| 趋势分析 | K线图 + 均线 |
| 对比分析 | 分组柱状图 |
| 结构分析 | 饼图 / 堆叠柱状图 |
| 异常检测 | 箱线图 |
| 相关分析 | 散点图 + 趋势线 |

## 金融图表设计规范

### 1. K线图规范

```
要素:
- 开盘价、收盘价、最高价、最低价
- 成交量（底部）
- 均线（MA5、MA10、MA20）

配色:
- 上涨: 红色 (#E74C3C)
- 下跌: 绿色 (#2ECC71)

标注:
- 关键价格位
- 支撑位/压力位
- 买卖信号
```

### 2. 趋势图规范

```
要素:
- 主线（粗线）
- 均值线（虚线）
- 置信区间（阴影）

配色:
- 主线: 蓝色 (#3498DB)
- 均值: 灰色 (#95A5A6)
- 区间: 浅蓝色

标注:
- 峰值/谷值
- 转折点
- 异常点
```

### 3. 对比图规范

```
要素:
- 分组柱状
- 数值标签
- 图例

配色:
- 使用对比色
- 保持一致性
- 避免过多颜色（≤5种）

标注:
- 数值标签
- 差异标注
- 排名
```

## 常见错误

### 1. 图表选择错误

❌ 错误: 用饼图展示时间序列
✅ 正确: 用趋势图展示时间序列

❌ 错误: 用折线图展示分类对比
✅ 正确: 用柱状图展示分类对比

### 2. 配色问题

❌ 错误: 使用过多颜色
✅ 正确: 控制在3-5种颜色内

❌ 错误: 红绿配色不统一
✅ 正确: 统一使用红涨绿跌

### 3. 信息过载

❌ 错误: 一张图展示太多信息
✅ 正确: 每张图聚焦一个主题

❌ 错误: 缺少标题和标签
✅ 正确: 清晰的标题、坐标轴标签

## 输出格式

### 图片格式

```
格式: PNG
分辨率: 100 DPI
尺寸: 
  - 默认: 800x400
  - 报告: 600x300
  - 大图: 1200x600
```

### 命名规范

```
格式: {类型}_{标识}_{时间戳}.png

示例:
- kline_600519_20240101_120000.png
- trend_stock_price_20240101_120000.png
- comparison_revenue_20240101_120000.png
```
